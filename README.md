# React useState Hook Practice

This project is a hands-on practice application built while learning React state management using the `useState` hook.

## Concepts Covered
- React component state
- useState hook basics
- Updating state using functions
- Re-rendering UI based on state changes
- Handling user interactions with event handlers

## What This Project Does
- Displays a dynamic value stored in React state
- Updates the UI when the state changes
- Demonstrates how React re-renders components automatically when state updates

## Technologies Used
- React
- JavaScript (ES6)
- JSX

## Learning Outcome
This project helped reinforce how React state works internally and how the `useState` hook is used to create interactive and dynamic user interfaces.
